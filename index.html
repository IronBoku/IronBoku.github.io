Excitecar: Panah kiri/kanan untuk pindah jalur.

Donkey Kong: Panah kiri/kanan untuk jalan, Space/Up/W untuk lompat.

Tetris: Panah kiri/kanan untuk geser balok, Panah bawah untuk turunkan cepat, Panah atas/W untuk rotate.

Dr. Mario: Panah kiri/kanan untuk geser kapsul, Panah bawah untuk turunkan cepat, Panah atas/W untuk rotate.
